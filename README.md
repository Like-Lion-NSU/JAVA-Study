## 📜 규칙
- 스터디는 디스코드로 매주 금요일 21시 진행
- repo는 fork해서 사용합니다.
- branch는 main만 사용합니다.
- 매주 각자 정해진 주제에 대해서 공부합니다.
- 모든 문서는 마크다운 형식을 사용하여 작성합니다.
- Repo에 공유는 매주 스터디 발표 전까지 완료합니다.
- 다른 폴더는 수정하거나 건들이지 않습니다.
- 발표자는 당일 랜덤으로 1명 선정합니다.
- 발표자 외 다른 스터디원들은 정리합니다.
- 질문은 pr을 통해 합니다.
- issue 제목은 주제로 합니다.
    - 제목 예시
        - 변수, 자료형, 연산자, 조건문, 반복문
- commit message 규칙
    - 예시
        - 변수, 자료형, 연산자, 조건문, 반복문 - #1
- 벌칙
